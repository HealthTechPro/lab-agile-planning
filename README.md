# Lab Agile Planning

## Overview

This repository was created as part of the IBM Introduction to Agile Development and Scrum course.

The purpose of this repository is to demonstrate Agile planning techniques using GitHub Projects and Kanban boards.

## Learning Objectives

- Agile Planning
- Scrum Framework
- Kanban Boards
- Product Backlog
- Sprint Backlog
- User Stories
- Agile Workflow Management

## Tools

- GitHub
- GitHub Projects
- Kanban
